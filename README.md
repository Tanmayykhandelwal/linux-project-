📌 Project Description

This project is part of the Open Source Software course. It involves analyzing an open-source software project, understanding its philosophy, license, ecosystem, and comparing it with proprietary alternatives.

Additionally, five Linux shell scripts are developed to demonstrate practical understanding of open-source tools and command-line operations.

📜 Script Descriptions
1. System Identity Report

Displays system information such as:

Kernel version
Logged-in user
System uptime
OS details
2. FOSS Package Inspector
Checks if a package is installed
Displays version, license, and summary
Uses case statement for description
3. Disk and Permission Auditor
Loops through important directories
Shows size, permissions, owner
4. Log File Analyzer
Reads log file line by line
Counts occurrences of a keyword (default: "error")
5. Open Source Manifesto Generator
Takes user input
Generates a personalized philosophy statement
Saves output to a .txt file
📌 Notes
All scripts are written in Bash
Each script includes comments for understanding
Tested on a Linux environment
